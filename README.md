# 减肥数据记录软件

这是一个简单的减肥数据记录应用，可以记录每天早晨和晚上的体重数据。

## 功能特点

- 记录每日早晨体重
- 记录每日晚间体重  
- 查看历史体重记录
- 数据按日期分组显示
- 详细历史记录查看
- 数据统计分析
- 中文字体支持

## 使用方法

1. 确保已安装Python 3.6或更高版本
2. 安装依赖包：`pip install -r requirements.txt`
3. 运行程序：`python main.py`

## 使用步骤

1. **记录体重**：
   - 点击"记录早晨体重"按钮记录早晨体重
   - 点击"记录晚间体重"按钮记录晚间体重
   - 在弹出的对话框中输入体重值（单位：斤）
   - 点击"保存"

2. **查看记录**：
   - 主界面显示最近记录列表
   - 点击左上角菜单按钮选择"查看详细历史"查看完整记录
   - 点击"数据统计"查看统计信息

3. **退出应用**：
   - 点击右上角退出按钮关闭应用

## 数据存储

所有数据将保存在程序同目录下的 `data.db` 文件中。

## 字体支持

应用使用项目内的中文字体文件。如果显示乱码：

1. 在程序目录创建 `fonts` 文件夹
2. 将中文字体文件 `simkai.ttf` 放入该文件夹
3. 重启应用

如果找不到 simkai.ttf，可以从以下位置获取：
- Windows 系统：C:/Windows/Fonts/simkai.ttf
- 网上下载免费中文字体
- 使用其他中文字体文件，并重命名为 simkai.ttf

## 注意事项

- 体重单位为"斤"（1斤=0.5公斤)
- 每天可以分别记录早晨和晚上两次体重
- 如果同一天同一时段重复记录，会更新原有记录
- 数据自动保存，无需手动保存
